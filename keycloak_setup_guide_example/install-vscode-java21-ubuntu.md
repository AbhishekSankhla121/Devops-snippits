# Install Java 21 (OpenJDK)

## Step 1: Open Terminal and Install Java 21

``` bash
sudo apt-get install openjdk-21-jdk
```

------------------------------------------------------------------------

## Step 2: Verify Java Installation

``` bash
java -version
```
